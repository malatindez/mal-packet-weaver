Detailed Description

template<typename PacketType>
class weaver::DerivedPacket< PacketType >

A templated class representing a derived packet from the base Packet class.

Derived packets are specialized implementations of Packet that define specific packet types. They inherit serialization and deserialization functionality from the base Packet class and can be used to encapsulate and manage different types of data for communication. The template parameter PacketType specifies the concrete derived packet type.

Template Parameters
PacketTypeThe specific derived packet type.
